## 🔌 Browser Extension

### [cors.extensions.prosofttools.com](https://github.com/ProSoftTools/cors.extensions.prosofttools.com)

A lightweight browser extension that enhances your browsing experience on [prosofttools.com](https://www.prosofttools.com). Supports **Chrome**, **Edge**, and **Firefox**.

**Key features:**
- ⚡ Zero configuration — install and it works
- 🔄 CORS bypass for seamless cross-origin requests
- 📡 SSE / streaming support
- 🔒 Restricted to `prosofttools.com` only

**Install:** Search for **"ProSoftTools Browser Extension"** in your browser's extension store, or [install manually from the latest GitHub release](https://github.com/ProSoftTools/cors.extensions.prosofttools.com/releases).

## 🤖 Copilot LLM Proxy

### [COPILOT-LLM-PROXY](https://github.com/ProSoftTools/COPILOT-LLM-PROXY)

A VS Code extension that bridges **GitHub Copilot's Language Model API** to an OpenAI-compatible REST API — with zero runtime dependencies.

**Key features:**
- 🔗 OpenAI-compatible `/v1/chat/completions` and `/v1/models` endpoints
- 🌊 Streaming (SSE) and non-streaming responses
- 🛠️ Tool calls / function calling
- 🧠 Thinking / reasoning tokens
- 🖼️ Image inputs (base64 data URIs)
- 🔑 Optional API key authentication
- 🚀 Auto-start on VS Code launch

**Quick start:**
```bash
# Install from VS Code Marketplace — search "Copilot LLM Proxy"
# Then start the server via Command Palette:
# Copilot LLM Proxy: Start Server
#
# Proxy runs at: http://localhost:4141/v1
```
